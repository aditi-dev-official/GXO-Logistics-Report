# 📌 Project Overview

The **GXO Logistics Performance Dashboard** is an interactive Business Intelligence solution developed using **Microsoft Power BI** to analyze warehouse operations, logistics performance, and supply chain efficiency.

The dashboard transforms operational logistics data into meaningful insights using **Power Query**, **Data Modeling**, and **DAX**, enabling managers and stakeholders to monitor warehouse KPIs, shipment performance, inventory movement, workforce productivity, and operational trends through interactive dashboards.

---

# 🎯 Project Objectives

- Monitor logistics performance
- Analyze warehouse operations
- Track shipment and order fulfillment
- Evaluate workforce productivity
- Monitor inventory movement
- Improve operational efficiency
- Support data-driven supply chain decisions
